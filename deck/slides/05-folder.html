RULE 1

在哪啟動，就在哪工作

Claude Code 只動這個資料夾——你的第二大腦有明確的邊界。

RULE 2

資料夾規則不是收納術

是告訴 AI 每份資料扮演什麼角色：講義唯讀、病例先去識別化、產出物集中管理。

RULE 3

用 Git 保護知識庫

不用懂指令——「出張嘴」叫 Claude Code 幫你 git init、commit，每一步都可回溯。
